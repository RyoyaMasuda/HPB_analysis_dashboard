# 画面設定のお願い
window環境において
- 画面サイズ：100%
- 解像度：1920×1080
- ブラウザの倍率：80〜90%

に設定をお願いします。<br>
※その他の環境では画面崩れが生じる可能性があります。適宜調整いただけますと幸いです。
